mysql語法

mysql基础

1.概念
数据库 存储数据的仓库，数据是有组织的进行存储
数据库管理系统 操纵和管理的大型软件
sql 操作关系型数据库的编程语言，定义了一套操作关系型数据库统一标准
2.mysql语法

(1)启动关闭
启动 net start mysq
关闭 net stop mysql
登录 mysql -u (用户名) -p (密码)
(2)分类
DDL 数据定义语言，用来定义数据库对象
DML 数据操作语言，用来对数据库表中的数据进行增删改
DQL 数据查询语言，用来查询数据库中表的记录
DCL 数据控制语言，用来创建数据库用户 控制数据库的访问权限

2.DDL
(1)查询所有数据库 show databases；
(2)查询当前数据库 select database();
(3)创建数据库 create database (需要创建的数据库名字)；
(4)删除数据库 drop database (需要删除数据库名字)；
(5)切换数据库 use (需要切换的数据库的名字)；

3.表操作
(1)创建表
use (需要创建表的数据库) 进入数据库
create table (表名)(
id int primary key auto_increment,
sage int ,
sname varchar(32) not null,
gender varchar(10) not null,
score double not null,
birthday date
(1)查询当前数据库所有表
先use sys 进入sys系统数据库
show tables；查询数据库所有表
(2)查看指定表结构
desc (需要查看的表名)
查询表数据
select * from course
查询函数
select要查的函数
user() 当前用户名
database() 当前所用数据库
current_user() D当前用户名查看权限
version() 数据库版本
@@datadir 数据库的路径

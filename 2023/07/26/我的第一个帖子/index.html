环境安装

1.提前安装好(node.js)[https://nodejs.org/en] 选左边的那个
2.提前安装好(GIt)[https://git-scm.com/downloads]
3.提前注册好一个github账号
4.安装完后 win+r 进入cmd 输入npm -v 有版本说明安装好了 再输入node-v 有版本也同样是安装好了

5..到此电脑右键鼠标，点击属性点击高级系统，点击环境变量

6.找到path 点编辑 把文件新建进去

操作步骤

1.进入到hexo的文件点右键，运行git
输入 npm install -g hexo-cli
hexo init blog
cd blog
没出错代表成功了
hexo new test_my_site 创建一个新的页面
hexo g 生成
hexo s 就可以复制链接去网站了 最开始的界面就出来了

2.配置ssh
​ cd ~/.ssh 检查密钥
​ ssh-keygen -t rsa -C “邮箱地址”
​ 再在git界面连续按三次回车键
​ 到个人设置查看密钥


回到git界面输入
ssh -T git@github.com 尝试ssh的连接
提示Are you sure you want to continue connecting (yes/no)? 输入yes
在文件夹找到hexo 进入blog文件的目录 按win+r 进入cmd
输入npm install hexo-deployer-git –save

返回到上一个hexo的git页面 继续配置
git config –global user.name “luozhixiaowo” 红色的地方写自己github的用户名
git config –global user.email “xxx@qq.com“ 红色的地方写自己当时注册github的邮箱

3.上传
到自己github主页 找到设置 点创建一个新的库

创建完成

进入hexo的git里 创建一个新的帖子 hexo -new “第一个帖子 “
hexo g 生成一下 hexo s 运行一下 最近hexo d 部署
